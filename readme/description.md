Описание проекта PSIG (Portable Streamer Interactive Game)

Игра делится на шесть основных компонентов:
- Интерфейс пользователя(frontend)
	- Сумка игрока
	- Карта
	- Таблица лидеров
	
- Интерфейс администратора(admin)

- Движок игры(backend)
	- интерактивная система ответов сервера
	- возможность подключения языковой модели
	- программы для выборки произвольного элемента в хранилище по определенным параметрам
	- броски кубиков; бросок монетки; 

- Правила игры(отдельная сущность для хранения ссылок на хранилище)
	- Описания всех сущностей. Возможность хранить в разных форматах
	- Предметы
	- Эффекты
	- Статусы игроков и администраторы
	- и т.д.
	
- API - интерфейс для подключения к движку из вне
	- api для фронта
	- api для общения с базой данных
	
- Хранилище:
	- база данных
	- временное, часто обновляемое хранилище

Игрок регистрируется на сайте и попадает в меню выбора карты.
На каждой карте лимит игроков. Выбрав карту, игрок получает доступ к карте и своему инвентарю.
Он видит страницу своего профиля. В нем описание, количество очков, позиция на карте и т.д.

Цель игры дойти до конца и выиграть в супер игре. Первые 3 места призовые.

У игрока есть возможность дать доступ стороннему аккаунту к своему профилю.

#maps_game 
Карта делиться на матрицу из чисел. К каждой клетке можно обратиться. У каждой клетки есть набор свойств(сам набор будет описан в структуре). Клетки могут скапливаться в биом(набор клеток). Есть регионы - таверны, для передышки игрока и зону безопасности от эффектов. Регионы могут делиться по типам, фильтрам, по свойствам биома(региона). Свойства биома могут взаимодействовать с игроком накладывая на профиль разные бафы и дебафы, взаимодействия меняющие направление игры.

У карты есть окно показывающее кто впереди. При наведении показывает сколько клеток +- до ближайшего игрока, кто он, бафы и дебафы, какие предметы из инвентаря можно использовать на него. Какие предметы в его рюкзаке можешь видеть ты.

Предметы делятся на группы. Могут накладывать эффекты. Могут давать очки прокачки. Инвентарь можно расширять за счет предметов. Их можно использовать, продать, обменять.

Ресурсы, предметы и эффекты можно посмотреть в собственном рюкзаке. В нем же и лежат предметы. Ресурсы накапливаются. Их можно тратить и получать за разные активности. Предметы через некоторое время могут взорваться. Динамит намокнуть и отсыреть если вы долго находитесь в биоме болото и т.д.

При наведении курсора на предметы высвечивается подсказка со ссылкой на правила игры.

Администратор редактирует правила игры и регулирует алгоритмы поведения сервера через интерфейс.

 Можно отдавать и получать запросы распараллелив несколько ядер и сразу используя асинхронный код для связи клиента с сервером.

Взаимодействия игроков меж собой описано в правилах. Они отражаются на карте как фигуры, аватары.
